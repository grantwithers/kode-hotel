Understanding Grid System

Back to Top

This template use bootstrap grid to make it very easy for anybody to understand the grid system. There is some basic details about its html structure below.

Parent Classes of Sections
.kf-home-banner This class is used for setting up the main banner

.kf-aboutus-wrap class is used for the about us in your site.

.kf-gallery-wrap This class is used for mesonery gallery .

.kf_content_wrap gallery This class is for showing the room grid page.

.kf-room-grid our-services This class is for showing the our services.

.ka-grid-layout-wrap This class is for the showing the detail view of our.

.kf-blog-list This class is for showing blog list page.

.kf_content_wrap blog-detail-page class is used for the blog details where you can show the blog in detailed view.

.kf-blog-3 class is used for third blog listing view.

.kf_comingsoon-wrap class is used for creating the coming soon page of your site.

.kf-404-wrap class is used for the error 404 page creation.
